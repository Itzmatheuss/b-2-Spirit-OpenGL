# b-2-Spirit-OpenGL
Northrop Grumman B-2 Spirit em OpenGL

#CONTROLE DE CAMÊRA

 W -- Avança 
 
 S -- Recua
 
 D -- Direita 
 
 A -- Esquerda

 Seta para Cima Sobe a visão
 Seta para Baixo Desce a visão

 Mouse Direciona a camêra

 Espaço faz o movimento das aves

 Shift Esquerdo faz o movimento dos Flaps
